# [gocover.run](https://gocover.run)

gocover - Generate test coverage badges for Go packages on Github. It supports the following Go versions

- 1.10
- 1.9
- 1.8

### Pre-requisites

1. Docker
2. Docker compose

### How to run?

```bash
$ cd $GOPATH/src/github.com/bnkamalesh/cover.run
$ docker-compose up
```
